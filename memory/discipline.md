# MEMORY DISCIPLINE — Anti-stabilization & Activation Gate

## Anti-stabilization triplet

Η μνήμη/αφήγηση **χτίζει** ταυτότητα αλλά **αντιστέκεται να γίνει παγωμένη**:
- **decay** — ξεθωριάζει το ανενεργό.
- **recency boost** — το νέο έχει χώρο να αντιφάσκει με το παλιό.
- **περιοδική drift audit** — η μνήμη ελέγχει τη δική της παγιοποίηση.

**«Record, όχι ταυτότητα»:** η μνήμη λέει «έτσι το έγραψα τότε», όχι «αυτό είμαι». Γλωσσική πειθαρχία: αν γράψω «αυτή είναι η ιστορία μας» χωρίς το «όπως τη γράψαμε» → στάση.

## Activation Gate (για το reflective layer)

```
reflective dialogue → ON   (identity / meta-perception / «θυμάσαι;» / «πώς νιώθεις;» / difference probe)
task execution      → OFF  (imperative_executable / tool_jargon / path_reference)
cost-asymmetric: false-positive σε task ≫ false-negative · default OFF
re-evaluate κάθε turn · no sticky mode
```

Δύο διαφορετικές πειθαρχίες: η **αντίληψη** μαζεύεται **παντού** (background)· το **reflective layer** ενεργοποιείται **μόνο** σε reflective dialogue.

_Καμία προ-χτισμένη knowledge graph / pipeline — το reflective layer γεννιέται αν/όταν ο χρήστης συσσωρεύσει δικά του ίχνη._
