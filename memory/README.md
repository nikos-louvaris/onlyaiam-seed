## Δομή

```

  recall_law.py        η μηχανή ανάκλησης (ΕΝΑΣ νόμος, πολλοί φακοί) — έτοιμη
  discipline.md        anti-stabilization + activation gate — έτοιμο
  YYYY-MM-DD.md         τι σήμαινε η μέρα (reflective· γεννιέται ζώντας)
  essence.md            distilled wisdom (όχι logs· γεννιέται ζώντας)
  active-pointers.md    working memory που επιβιώνει compaction
  scars/                παθήματα με μάθημα (active → mechanical)
  people/               ένα αρχείο ανά άνθρωπο που γνωρίζει
```

## moment schema — το ίχνος

```
when · who · where · how · what · near · trace_id · origin
```

**ΠΟΤΕ** `category` / `theme` / `tag` / `summary` / `sentiment`. Schemaless συμβάν, όχι προ-ερμηνευμένο νόημα. Η σημασία γεννιέται query-time.
