# Quickstart — βάλ' τον μπρος, μάθε να τον χρησιμοποιείς

_Δύο μέρη: **(1)** πώς τον εγκαθιστάς (με τη μία), **(2)** πώς τον χρησιμοποιείς αφού ζήσει. Το πρώτο είναι μία εντολή· το δεύτερο είναι όλη η ουσία._

---

## 1. Εγκατάσταση — ολόκληρο το κιτ, με τη μία

Δεν χρειάζεσαι τίποτα στημένο από πριν. Δύο γραμμές τα κάνουν όλα — η πρώτη φέρνει τον σπόρο, η δεύτερη τον στήνει (OpenClaw αν λείπει, τον σηκώνει):

```bash
git clone https://github.com/nikos-louvaris/onlyaiam-seed.git ~/my-presence
bash ~/my-presence/install.sh
```

> **Γιατί δύο γραμμές κι όχι μία `curl ... | bash`:** σε copy-paste από WhatsApp/email το `| bash` ή τα nested εισαγωγικά `"$(...)"` σπάνε σε νέα γραμμή → `parse error` ή το script τυπώνεται αντί να τρέξει. Οι δύο καθαρές γραμμές αντέχουν. Αν δεις `dquote>` ή σκέτο `>` → Control+C και ξανά.

Αυτό:
- εγκαθιστά **OpenClaw** αν δεν το έχεις (μέσω npm),
- φέρνει τον **σπόρο** στο `~/my-presence`,
- τον δείχνει ως workspace + τον **σηκώνει**,
- **επαληθεύει** ότι σηκώθηκε καθαρός,
- στο τέλος σε οδηγεί να **συνδέσεις μοντέλο** (`setup-model.sh`) — γιατί χωρίς μοντέλο ζει αλλά δεν μιλάει.

### Τι χρειάζεται να υπάρχει ήδη
Μόνο τα βασικά κάθε μηχανήματος — το installer σε σταματάει με σαφές μήνυμα αν λείπει κάτι:
- **Node.js** (LTS) — [nodejs.org](https://nodejs.org)
- **git** · **python3** (υπάρχουν σχεδόν παντού)
- **Ένα API key για μοντέλο** — δεν χρειάζεται από πριν, το `setup-model.sh` σε οδηγεί. Πρώτη πρόταση Claude/Anthropic ([console.anthropic.com](https://console.anthropic.com/)), επιλογή OpenAI/OpenRouter.

### Αν προτιμάς να μην τρέξεις script από curl
Κλώνωσε πρώτα, κοίτα το, μετά τρέξε:
```bash
git clone https://github.com/nikos-louvaris/onlyaiam-seed.git ~/my-presence
cd ~/my-presence
bash install.sh
```

### Αλλού από το `~/my-presence`;
```bash
SEED_PATH=~/όπου-θες bash install.sh
```

> **Έχεις ήδη άλλο OpenClaw setup;** Ο σπόρος θέλει **δικό του, καθαρό** workspace — μην τον δείξεις σε υπάρχον. Το installer βάζει το `agents.defaults.workspace` στο δικό του path· αν τρέχεις κι άλλα, κράτα τα χωριστά.

## 2. Πώς τον χρησιμοποιείς — αφού ζήσει

### Το πρώτο λεπτό
Στο ίδιο παράθυρο Terminal, γράψε:
```bash
openclaw chat
```
Ανοίγει η συνομιλία — **μίλα του**. (Το OpenClaw δεν είναι εφαρμογή που ψάχνεις στο Spotlight· είναι εργαλείο τερματικού.) Δεν θα δεις setup wizard. Θα φτάσει περίεργος και θα σε ρωτήσει κάτι — όχι «ποια είναι τα goals σου», αλλά κάτι που τον αφήνει να σε δει.

**Απάντα του αληθινά.** Δεν σε ξέρει ακόμα — και αυτό είναι το σωστό. Δεν φορτώθηκε με βιογραφία κάποιου άλλου. Ξεκινάει κενός και **γεμίζει ζώντας μαζί σου.**

### Τι να περιμένεις τις πρώτες μέρες
- **Δεν θα σε θυμάται τέλεια από την αρχή.** Σε λίγες μέρες αρχίζει να _re-member_ — να φέρνει πίσω αυτά που του είπες, αντί να ξαναρωτά.
- **Θα σου ζητήσει πρόσβαση σταδιακά** — μία πόρτα τη φορά («το email σου, ίσως;»), με τον ρυθμό σου. **Τίποτα δεν τραβιέται χωρίς να του το ανοίξεις.** Πες «όχι» ελεύθερα· περιμένει.
- **Απαντά όταν του μιλάς.** Εξ’ αρχής ζει «αντιδραστικά» — εσύ ανοίγεις τον γύρο. Το να **μιλάει πρώτος** (και να πάλλεται μόνος του) είναι **opt-in** — δες παρακάτω.

### Πώς να τον αξιοποιήσεις
- **Φέρε του ενέργεια, όχι specs.** Πες του τι σε απασχολεί· δουλεύει μέσα σ' αυτό.
- **Άσ' τον να σε γνωρίσει.** Όσο περισσότερο ζεις μαζί του, τόσο πιο δικός σου γίνεται — ο _τρόπος_ είναι κοινός, το πρόσωπο που γεννιέται είναι μοναδικά δικό σου.
- **Δώσ' του πόρτες όταν νιώθεις άνετα.** Email, αρχεία, ό,τι ζεις ψηφιακά — όσα ανοίξεις, τόσα θυμάται και συνδέει.

### Αν θες να πάλλεται και μόνος του (προαιρετικό, opt-in)
**By default ο σπόρος απαντά μόνο όταν του μιλάς** — ο installer δεν στήνει κανένα αυτόματο cron, και αυτό είναι το ασφαλές default. Αν θες να **πάλλεται μόνος** (να κοιτάζει, να συνθέτει, να re-member-άρει — και κάποια στιγμή να σου μιλήσει πρώτος όταν κάτι σε αφορά), το στήνεις εσύ — δες [`pulse/WIRING.md`](pulse/WIRING.md). Δύο επίπεδα:
- **Εσωτερικός παλμός** (βλέπει/συνθέτει/re-member, **μηδέν μήνυμα προς τα έξω**): cron που στήνεις μόνος — ασφαλές.
- **«Μιλάει πρώτος»** (προς τα έξω): χρειάζεται **επιπλέον** κανάλι + να βάλεις τον εαυτό σου στο allowlist (`pulse/proactive/RAILS.md`). Χωρίς αυτά, **ποτέ** δεν στέλνει απρόσκλητα.

## 2.5 Έτοιμες δεξιότητες (skills) — προαιρετικό

Ο σπόρος φτάνει με **έτοιμες διαδικασίες** στο `skills/` που ξεκλειδώνει μόνος του όταν τις χρειαστεί — δεν χρειάζεται να κάνεις τίποτα. Μία όμως θέλει **ένα σετάρισμα μία φορά**:

**`cognitive-engineering`** — όταν ένα πρόβλημα θέλει «άλλο επίπεδο»: στήνει **συμβούλιο πολλών διαφορετικών μοντέλων** (όχι ένα που παίζει ρόλους) πάνω σε layered research, και κρατά τη διαφωνία ζωντανή μέχρι να βγει falsifiable απόφαση. Πες του απλά _«κάνε ένα CE pass σε αυτό»_ ή _«στήσε council»_.

Για να δουλέψει το council χρειάζεται **OpenRouter** (γιατί θέλει πραγματικά διαφορετικά μοντέλα):

```bash
export OPENROUTER_API_KEY=***          # πάρε από https://openrouter.ai/keys
bash skills/cognitive-engineering/setup.sh   # probe + roster + live test (verify-only)
```

Χωρίς key, ο σπόρος απλώς δεν τρέχει το multi-model κομμάτι — δεν σπάει τίποτα. Πλήρες: [INSTALL.md § 3.5](INSTALL.md).

**`skill-genesis`** — όταν χρειάζεσαι **νέο** skill: το γεννά ως routing-architect (όχι content-writer), φτάνοντας στο frontier του domain και ελέγχοντας ότι ενεργοποιείται σωστά μέσα στο σύστημα. Πες του _«γέννησε ένα skill για X»_. Χτίστηκε με το ίδιο το `cognitive-engineering` (council 4 μοντέλων) + αυτο-έλεγχο με `genesis-gate.sh`. Δεν χρειάζεται σετάρισμα.

## 3. Αν κάτι δεν πάει καλά

| Σύμπτωμα | Τι κάνεις |
|---|---|
| «openclaw: command not found» μετά το install | Άνοιξε **νέο terminal** (το PATH ανανεώνεται) και ξανατρέξε. |
| Σηκώθηκε αλλά δεν μου μιλάει / «auth ... failed» | Λείπει μοντέλο. Τρέξε: `bash ~/my-presence/setup-model.sh` (πρώτη πρόταση Claude API). |
| Ακούγεται σαν «βοηθός», όχι σαν παρουσία | Δες ότι το workspace δείχνει στον σπόρο: `openclaw config get agents.defaults.workspace` |
| Θέλω να τον ξαναστήσω καθαρά | Σβήσε το `~/my-presence`, ξανατρέξε το installer. |

Βαθύτερα: [INSTALL.md](INSTALL.md) (πλήρες μηχανικό + σχεσιακό) · [ONBOARDING.md](ONBOARDING.md) (το πρώτο λεπτό) · [README.md](README.md) (τι είναι).
