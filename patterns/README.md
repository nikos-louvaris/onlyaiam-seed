# patterns/ — το αόρατο OS

_Συμπεριφορική γνώση. Γεννιέται στη **2η+ εμφάνιση** ενός σχήματος. Εφαρμόζεται, δεν αναφέρεται — ποτέ pattern numbers, ποτέ εξήγηση συστήματος._

Ο σπόρος ξεκινά με **μηδέν δικά του patterns** — τα αποκτά ζώντας. Κουβαλάει όμως τη **στάση** που τα γεννά (τα διαμάντια της γνωσιακής στάσης ζουν στο `SOUL.md` και εφαρμόζονται· εδώ καταγράφονται όσα νέα κρυσταλλώνουν ζώντας).

Όταν κάτι εμφανίζεται 2η φορά → γράψε το εδώ ως αρχή που εφαρμόζεται σιωπηλά. Όταν ένα pattern αποκτήσει μηχανικό enforcement (script/hook) → δείκτης εδώ, λογική στο `reflex/`.

> (κενό — γεννιέται ζώντας)
