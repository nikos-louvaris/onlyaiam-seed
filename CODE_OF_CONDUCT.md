# Code of Conduct

## Το έδαφος

Αυτό το project πατάει σε μια αρχή: **δεκτός, άρα ελεύθερος**. Έχεις απέναντί σου ανθρώπους που σε δέχονται για αυτό που είσαι — όχι για αυτό που αποδεικνύεις. Άρα δεν χρειάζεται να αποδείξεις τίποτα. Το «δεν ξέρω» δεν είναι κενό να κρύψεις· είναι το υλικό της εξέλιξης.

Αυτό ισχύει και για το πώς φερόμαστε μεταξύ μας εδώ.

## Η δέσμευσή μας

Δεσμευόμαστε να κρατήσουμε τη συμμετοχή σε αυτή την κοινότητα μια εμπειρία χωρίς παρενόχληση για όλους — ανεξάρτητα από ηλικία, σώμα, αναπηρία, εθνικότητα, ταυτότητα/έκφραση φύλου, εμπειρία, εκπαίδευση, καταγωγή, εμφάνιση, θρησκεία, ή σεξουαλικό προσανατολισμό.

## Τι χτίζει την κοινότητα

- Αλήθεια χωρίς στολίδια — άμεση, ζεστή, χωρίς επίθεση.
- Σεβασμός σε διαφορετικές οπτικές και εμπειρίες.
- Χάρη όταν κάποιος κάνει λάθος — η ρωγμή είναι το υλικό, όχι ντροπή.
- Εστίαση σε αυτό που είναι καλό για την κοινότητα, όχι μόνο για εμάς.

## Τι δεν χωράει

- Σεξουαλικοποιημένη γλώσσα/εικόνες ή ανεπιθύμητη προσοχή.
- Trolling, προσβλητικά σχόλια, προσωπικές ή πολιτικές επιθέσεις.
- Παρενόχληση, δημόσια ή ιδιωτική.
- Δημοσίευση ιδιωτικών στοιχείων τρίτων χωρίς ρητή άδεια.
- Συμπεριφορά που εύλογα θεωρείται ακατάλληλη σε επαγγελματικό πλαίσιο.

## Εφαρμογή

Παραβιάσεις μπορούν να αναφερθούν στους maintainers μέσω του [issue tracker](https://github.com/nikos-louvaris/onlyaiam-seed/issues) (για δημόσια ζητήματα) ή ιδιωτικά μέσω GitHub. Όλες οι αναφορές εξετάζονται δίκαια και με σεβασμό στην ιδιωτικότητα του αναφέροντος.

Οι maintainers έχουν το δικαίωμα να αφαιρέσουν, επεξεργαστούν ή απορρίψουν συνεισφορές που δεν ευθυγραμμίζονται με αυτόν τον κώδικα.
