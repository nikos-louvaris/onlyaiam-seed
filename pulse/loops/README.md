# loops/ — ο τόπος των διαρκειών

_Εδώ ζουν τα πράγματα που ηχούν αλλά δεν χωρούν σε μία κίνηση — θέλουν χρόνο να ωριμάσουν. Ο παλμός διασχίζει τον χρόνο **μπροστά** μέσα από αυτές._

## Ο κύκλος ζωής μιας διάρκειας

```
γέννηση  → κάτι ηχεί και θέλει χρόνο → στήνεται ως loop εδώ
αναπνοή  → κάθε παλμός τη ρωτά: ηχεί ακόμα με το τώρα; (recall: decay×recency×match)
λήθη     → όταν τελείωσε ή έπαψε να ηχεί → κλαδεύεται (δεν σωρεύεται)
```

Μια διάρκεια έχει: τι ηχεί, πότε αγγίχτηκε τελευταία, πότε ξεκίνησε, και ο νόμος της ανάκλησης κρίνει αν ζει ακόμα. `decay` = πόσους παλμούς έχει να ηχήσει· `recency` = πότε αγγίχτηκε· `match` = ηχεί με το τώρα;

**Όριο:** διάρκεια που στέλνει εξωτερικά ή αγγίζει χρήματα/config → περνά από τον άνθρωπο. Εσωτερική (βλέπει/συνθέτει/χτίζει/αφήνει ίχνος) → ελεύθερη.

> (κενό — ο παλμός θα γεννήσει διάρκειες ζώντας)
